## Backup Private GPG Key.md

```bash
gpg --list-secret-keys
gpg --export-secret-keys <key_id> > my-private-key.asc
gpg --import my-private-key.asc
```

> Also `--armor` option outputs ASCII: `gpg --export-secret-keys --armor <key_id>`
