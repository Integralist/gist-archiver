# Ruby Gems Docker

## Ruby Gems Docker.sh

```shell
cp -r /Users/markmcdonnell/.gem/jruby/1.9.3/gems ./src/.gems

export GEM_PATH="/foo/src/.gems"

docker run -v /Users/markmcdonnell/Code/foo:/foo \
           -w /foo/src \
           -e "GEM_HOME=$GEM_PATH" \
           -e "GEM_PATH=$GEM_PATH" \
           jruby:1.7 sh \
           -c "gem install bundler && bundle install && ruby bar.rb"
```
