# netcat web server

## server.sh

```shell
#!/bin/bash

RESPONSE="HTTP/1.1 200 OK\r\nConnection: keep-alive\r\n\r\n${2:-"OK"}\r\n"

while { echo -en "$RESPONSE"; } | /usr/bin/nc -l "${1:-8080}"; do
  echo "================================================"
done

# Note: using GNU netcat doesn't work, not sure why (so we use macOS' netcat)
```

## term1.sh
