## Vim programming autocomplete with no plugins.md

```viml
filetype plugin on
set omnifunc=syntaxcomplete#Complete
```

`<C-x><C-o>`
  
Vim also provides the native command `<C-n>` for autocompletion based on words existing in all opened files (buffers).
