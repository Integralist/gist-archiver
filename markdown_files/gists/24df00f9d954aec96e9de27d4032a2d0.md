## gpg agent connection refused.md

Error:

```
can't connect to `/root/.gnupg/S.gpg-agent': Connection refused
gpg: can't connect to `/root/.gnupg/S.gpg-agent': connect failed
```

Try running:

```bash
$ gpg-connect-agent /bye
```
