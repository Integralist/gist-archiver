## Bash Print All Alias.sh

```shell
bold=$(tput bold)
normal=$(tput sgr0)

# list out all custom defined alias'
alias list='cat ~/.bashrc | grep "^alias" | gsed -En "s/alias (\w+)=(.+)/${bold}\1\n  ${normal}\2\n/p"'
```
