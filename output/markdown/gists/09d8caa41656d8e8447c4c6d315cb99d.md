## Build Tags

File: `a/a.go`

```go
// +build !mock

package a
func DoSomething() {
    return "done"
}
```

File: `a/a_mock.go`

```go
// +build mock

package a
func DoSomething() {  // Insert fake implementation here
    return "complete"
}
```

Execute tests using build tag:

```bash
$ go test -tags mock
```
