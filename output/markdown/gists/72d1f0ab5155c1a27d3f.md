# Ruby Mock Redis

## Ruby Mock Redis.rb

```ruby
# in your spec helper...

require "pry"
require "redis"
require "mock_redis"

RSpec.configure do |config|
  config.before(:each) do
    mock_redis = MockRedis.new
    allow(Redis).to receive(:new).and_return(mock_redis)
  end
end

# Also you can do...

require 'mock_redis'
>> mr = MockRedis.new
>> mr.set('some key', 'some value')
=> "OK"
>> mr.get('some key')
=> "some value"
```
