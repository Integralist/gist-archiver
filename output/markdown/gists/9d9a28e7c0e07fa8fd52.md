# POST data via curl command using CERT and password

## curl and cert password.bash

```shell
curl --verbose \ 
     --header "Content-Type: application/json" \
     --request POST \
     --data '{"foo":"bar"}' \
     --insecure \
     --cert $DEV_CERT_P12:{password} \
     https://domain.com/path/to/endpoint
```
