[Base64 JS Object] Allow for passing complex object to an iframe via Query String

View original Gist on GitHub

Tags: #js #pickle #base64

base64 js object.js

var s = window.btoa(JSON.stringify({'id': 1234}));
// "eyJpZCI6MTIzNH0="

var o = JSON.parse(window.atob(s));
// {id: 1234}

o.id
// 1234
